# Saucedemo-QA-Testing-Project
Comprehensive QA testing (manual, exploratory &amp; automation) on the SauceDemo application.

## Project Overview
This repository contains structured QA activities performed on the SauceDemo application, including:
- Manual test case design
- Bug documentation
- Exploratory testing
- Session & state validation testing
- Cross-user behavioral testing
- Automation Testing with Selenium + Java (In progress)

## Application Under Test
https://www.saucedemo.com

- problem_user
- blocked_user

## Project Structure
- Manual Testing (Test Cases, Bug Reports, Exploratory Report)
- Automation Testing (In Progress)
- Screenshots & Screenrecords (Test Evidence)
